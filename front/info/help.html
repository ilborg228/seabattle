    Автоматизированная система "игра "Морской бой"

    Введение

    Автоматизированная система "игра "Морской бой" является развлекательной программой, переносящей знаменитую настольную игру в электронный формат. Она позволяет пользователю играть как против ИИ, так и против других игроков
    Программа является клиент-серверным приложением. Серверная часть использует базу данных MySQL(требует предустановления сервера MySQL и базы данных) и среду выполнения JS кода NodeJS. Программная система совместима с Windows XP, Windows Vista. Системные требования: 150 Mb оперативной памяти, 25 Mb жесткого диска.

    Запуск программы

    При запуске программной системы появляется окно, представленное на рисунке. В поле для ввода никнейма необходимо ввести под каким именем пользователь будет виде другим игрокам и нажать кнопку "Войти".



    Если данные введены верно, то система разблокирует все функции, предоставленные игроку, что продемонстрировано на следующем рисунке.



    Функции, предоставленные игроку

    Основными функциями игрока являются:

    • ручная расстановка кораблей;
    • выбор стратегии для автоматической расстановки кораблей;
    • сохранение игрового поля в файл;
    • загрузка игрового поля из файла;
    • выбор сложности для игры против ИИ;
    • сетевая игра;
    • включение/выключение звуковых эффектов;
    • включение/выключение фоновой музыки.

    Расстановка кораблей

    Ручная расстановка

    Для того, чтобы поставить корабль на поле, необходимо навести курсор на нужный корабль, потом зажать левую кнопку мыши и перетащить корабль на игровое поле, изменяя его направление колесиком мыши. Если выбранное место не подходит, корабль вернется на исходную позицию

    Автоматическая расстановка

    Для автоматической расстановки кораблей необходимо выбрать стратегию (случайная, "Берега", "Диагонали") и нажать кнопку "Расставить корабли по стратегии".



    Сохранение расстановки

    Для того, чтобы сохранить расстановку, необходимо ввести название расстановки в нужном поле и нажать кнопку "Сохранить файл расстановки", после чего файл окажется в папке "Загрузки", что продемонстрировано на рисунках.




    Загрузка расстановки

    Для того, чтобы загрузить файл расстановки, необходимо нажать кнопку "обзор" в стандартной форме загрузки, после чего откроется окно, где можно выбрать нужный файл. Если файл не подходит или поврежден, будет выдано соответствующее сообщение.



    Очистка поля и удаление кораблей

    Чтобы очистить поле, необходимо нажать кнопку "убрать корабли". Если пользователь захочет убрать только один корабль, достаточно перетащить его в область под игровым полем.

    Игровой процесс

    Игра против ИИ

    Для того, чтобы сыграть с искуственным интеллектом, необходимо сначала расставить корабли, затем выбрать уровень сложности и нажать кнопку "Играть против ИИ", изображенной на рисунке.



    Во время игры против ИИ необходимо атаковать клетки поля справа при помощи мыши. Точками обозначаются промахи. Желтое пламя - палуба раненного корабля, красное памя - палуба убитого корабля. Игровой процесс продемонстрирован на рисунке.



    Игрок может в любой момент сдаться, и вернуться на главный экран, тогда ему будет засчитано поражение.

    Игра против другого игрока

    Отличием от игры против ИИ является то, что перед началом боя ведется поиск противника, что продемонстрировано на рисунке. В остальном, игровой процесс не отличается от игры против ИИ.



    Настройка звука

     В программе присутствует фоновая музыка и звуковые эффекты. Включить или выключить их можно при помощи кнопок слева вверху, которые продемонстрированы на рисунке.



    Выход из программы

    Осуществляется выход из программы, ее полное завершение.
